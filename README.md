# Introduction

Welcome to DnDnDB, a web app for building static pages with a drag and drop interface. We are using React with Typescript on the front end and Django REST Framework with MySQL on the back end. Please [see the front end repo here](https://github.com/chingu-voyages/V54-tier3-team-38).
